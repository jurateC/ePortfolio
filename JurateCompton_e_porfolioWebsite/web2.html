Web Development2
with Helen Williams

In Web Development 2 module we built on knowledge learned in the last semester. We learned more about HTML 5 tags, block and inline elements, generic elements, the box model, grids and navigation. We also learned more about how to control layout with CSS, how to create and style tables and HTML forms.
For the Web Development 2 assignment we were required to create an ePortfolio website showcasing our work so far, styled using HTML5 and CSS3. We also had to test it and host it. We also had to write a report to summarize what we did, what went well, what was not working as we wanted and what we would do differently.
